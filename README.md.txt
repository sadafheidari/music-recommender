# Spreadsheet Automation Project

A Python automation project that streamlines Excel spreadsheet processing using the OpenPyXL library.

## Overview

This project demonstrates how Python can automate common spreadsheet tasks, reducing manual work and improving efficiency when handling Excel files.

The application reads transaction data from an Excel workbook, processes the information programmatically, and saves the updated spreadsheet.

## Features

- Read data from Excel (.xlsx) files
- Process spreadsheet records automatically
- Modify and update worksheet values
- Save changes to the workbook
- Eliminate repetitive manual Excel operations

## Technologies

- Python
- OpenPyXL
- Microsoft Excel (.xlsx)

## Project Files

- ExcelSpreadsheetsProject.py
- transaction.xlsx

## Installation

Install OpenPyXL:

```bash
pip install openpyxl
```

## Run

```bash
python ExcelSpreadsheetsProject.py
```
